# forumBiznesowe
Projekt zespołowy na zajęcia z Programowania Aplikacji Biznesowych w oparciu o Platformę Java. Celem było stworzenie forum z dodawaniem, wyświetlaniem, edytowaniem i komentowaniem postów. 
<p>API - SpringBoot</br>
Frontend - Angular7+</p>
